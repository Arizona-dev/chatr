## Usage


## Client

```bash
(before do: "make start" to up your containers)
make client
```

## Server

```bash
(before do: "make start" to up your containers)
make server
```

## Restart containers
```bash
make restart
```

## Migrate
```bash
make migrate
```

## Fixtures
```bash
make fixtures
```

## Production

```bash
make production
```

## Run the production server

```bash
make run
```
